汇编语言学习(1)

3月 4, 2020

1.1 机器语言

机器语言就是机器指令的集合

1.2 汇编语言的产生

汇编语言的主体是汇编指令

举个例子
机器指令: 1000100111011000
操作: 寄存器BX的内容送到AX中
汇编指令: MOV AX,BX

寄存器：简单地讲是cpu中可以存储数据的器件，一个cpu有多个寄存器。
AX 是其中一个寄存器的代号，BX 是另一个寄存器的代号。

1.3 汇编语言的组成

  1. 汇编指令 (机器码的助记符) 核心
  2. 伪指令 (有编译器执行)
  3. 其他符号 (有编译器识别)

1.4 存储器

计算机大部分部件都存在存储器
指令和书记在存储器中存放，也就是平常所说的内存

1.5 指令和数据

在内存或磁盘中，指令和数据是没有任何区别的，都是二进制信息。
举个例子
1000100111011000 -> 89D8H(数据)
1000100111011000 -> MOV AX,BX(程序)

1.6 存储单元

存储器被划分为若干个存储单，每个存储单元从0开始顺序编号
举个例子
一个存储器有128个存储单元，则编号从0~127.
存储单位：
1KB = 1024B
1MB = 1024KB
1GB = 1024MB
1Tb = 1024GB

1.7 CPU对存储器的读写

CPU要想进行数据的读写，必须和外部期间进行三类信息的交互：

  1. 存储单元的地址 (地址信息)
  2. 器件的选择,读或写 (控制信息)
  3. 读或写的数据 (数据信息)

计算机连接CPU和其他芯片的导线，通常称为总线。
逻辑上可划分为：

  1. 地址总线
  2. 数据总线
  3. 控制总线
